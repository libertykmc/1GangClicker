# 1GangClicker
Цели проекта:

Создание мобильного приложения в жанре кликер-игры с простым и увлекательным игровым процессом, направленным на постепенное развитие персонажа.
Разработка системы внутриигровых ресурсов (деньги, энергия и т.п.) и механик их получения и расходования.
Обеспечение удобного интерфейса и навигации через меню
Реализация базовых элементов прогрессии – улучшения, скины, достижения
Подготовка приложения к дальнейшему расширению функционала с интеграцией бекенда, системой регистрации/авторизации и сохранением данных в облаке. Задачи проекта:
Разработать главный экран с персонажем, реагирующем на клики: • Начисление денег • Уменьшение энергии
Создать окно магазина, в котором игрок может: • Покупать улучшения • Приобретать скины персонажа
Разработать страницу гардероба, где отображаются купленные игроком варианты внешнего вида.
Реализовать профиль игрока с системой достижений.
Обеспечить интуитивную навигацию по разделам.
Спроектировать модуль для дальнейшей интеграции с сервером: • Сохранение прогресса • Авторизация и регистрация персонажей
